Sumo WordPress theme
----------------------
Sumo theme is a Responsive WordPress Omega child theme. Utilizing hand crafted HTML5 code, layout options, customizer, big fat footer widget, social profile links.

Created by sumothemes.com, http://sumothemes.com

Install
-------
1. Download "omega" parent theme from http://wordpress.org/themes/omega
2. Go to your WordPress dashboard and select Appearance > Install, click Upload link.
3. Upload both Omega and this child theme, then activate the child theme

Setting up the social menu
--------------------------
1. Go to your WordPress dashboard and select Appearance > Menus, click 'create a new menu'
2. Click "Links" tab, input social URL, Link Text and Add to Menu
3. Choose Theme locations 'Social'
